data_access

Functionality to access Euclid calibrated frames, mosaic tiles and raw data for specific targets and observations.

source

DataAccess


def DataAccess(
    esa_username:NoneType=None, # ESA account username (prompts if not supplied)
    esa_password:NoneType=None, # ESA account password (prompts if not supplied)
    release_name:str='DR1', # Q1 or DR1
    esac_server_url:NoneType=None, # ESA server (default is to choose based on release_name)
    dry_run:bool=False, # if True, do not actually download files
    overwrite:bool=False, # should existing files be overwritten?
    science_only:bool=True, # if True, only return science frames or observations containing them
):

Provides access to Euclid data.

Example

The following demonstrates use of the DataAccess class. This would normally first be imported using from nicl.euclid.data_access import DataAccess.

To actually download files, remove dry_run=True.

release_name = "DR1"

da = DataAccess(release_name=release_name, dry_run=True)

We will download the files into the folder structure at the default location.

outpath = default_data_path(release_name)
